Buttons

The Nova Design System offers two distinct button styles: filled and outlined. Each style is suited for specific use cases, allowing for versatile application across different scenarios.

Filled Button

HTML
<button type="button" class="btn btn-style-filled">Filled Button</button>
CSS
.btn-style-filled {
  color: #ffffff;
  background-color: #096dd9;
  border: 0;
  font-weight: 400;
  border-radius: 8px;
  font-size: 16px;
  padding: 8px 16px;
}
.btn-style-filled:hover {
  background-color: #0050b3;
}

Outlined Button

HTML
<button type="button" class="btn btn-style-outlined">Outlined Button</button>
CSS
.btn-style-filled {
  color: #096dd9;
  background-color: #ffffff;
  border: 2px solid #096dd9;
  font-weight: 400;
  border-radius: 8px;
  font-size: 16px;
  padding: 8px 16px;
}
.btn-style-filled:hover {
  color: #ffffff;
  background-color: #096dd9;
}
