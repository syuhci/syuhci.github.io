Card

The Nova Design System features a card component that serves as a versatile and expandable container for content. It boasts a modern and straightforward appearance, enhanced by shadows for depth. This component is designed for high customizability to suit various needs.

Default Card

Card Title

Some quick example text to build on the card title and make up the bulk of the card's content.

Go somewhere
HTML
<div class="card-style-default shadow-sm border-0">
  <div class="card-body">
  <img src="image-holder.png" class="card-style-default-img-top mb-3" alt="image holder"/>
  <p class="card-title text-capitalize h5">Card Title</p>
  <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
  <a href="#" class="btn btn-primary btn-style-filled">Go somewhere</a>
</div>
CSS
.card-style-default {
  border-radius: 16px;
}
.card-style-default:hover {
  box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.08) !important;
  transform: scale(1.05);
  transition: transform 0.2s ease;
}
.card-style-default-img-top {
  border-radius: 8px;
}
